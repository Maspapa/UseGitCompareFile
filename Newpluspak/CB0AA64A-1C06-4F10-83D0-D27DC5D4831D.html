ラジオボタンの拡大

CheckMarkSize プロパティでラジオボタンのサイズを設定できます。CheckMarkSize プロパティの設定を有効にするには StretchSystemCheckMark プロパティに True を設定します。


gcRadioButton1.StretchSystemCheckMark = true;
gcRadioButton1.CheckMarkSize = new Size(20, 20);
GcRadioButton1.StretchSystemCheckMark = True
GcRadioButton1.CheckMarkSize = New Size(20, 20)
