チェックマークの拡大

CheckMarkSize プロパティでチェックマークのサイズを設定できます。CheckMarkSize プロパティの設定を有効にするには StretchSystemCheckMark プロパティに True を設定します。


gcCheckBox1.StretchSystemCheckMark = true;
gcCheckBox1.CheckMarkSize = new Size(20, 20);
GcCheckBox1.StretchSystemCheckMark = True
GcCheckBox1.CheckMarkSize = New Size(20, 20)
