GcRadioGroupBox コントロールは CheckMarkSize プロパティでチェックマークのサイズを設定できます。

```csharp
gcRadioGroupBox1.ItemSettings.CheckMarkSize = new Size(20, 20);
```

```vbnet
GcRadioGroupBox1.ItemSettings.CheckMarkSize = New Size(20, 20)
```
